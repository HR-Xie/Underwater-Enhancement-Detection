The model training code is located in the "Train" directory, and the dataset can be downloaded to a directory at the same level as "Train".
data1.yaml corresponds to the UOD dataset. data2.yaml corresponds to the DUO dataset.
The code for DLGA and DPC can be viewed in the "ultralytics/nn/ours" folder. The path to LUOD-YOLO.yaml is "ultralytics/cfg/models/v8/LUOD-YOLO.yaml". The paths in data.yaml and the training code need to be modified according to your actual path. We have recently made improvements to some details of the model to optimize its performance, and we may continue to develop and optimize this model in the future.
